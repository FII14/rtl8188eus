## Instal Driver Wifi Realtek RTL8188EUS di Kali Linux

```sh
$ sudo apt-get update
$ sudo apt-get upgrade
$ reboot
$ sudo apt-get install git
$ git clone https://github.com/FII14/rtl8188eus
$ cd rtl8188eus
$ chmod +x instal.sh
$ ./instal.sh
```
