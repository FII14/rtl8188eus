## Instal Driver Wifi Realtek RTL8188EUS di Kali Linux

```sh
apt-get update
apt-get upgrade
reboot
apt-get install git
git clone https://github.com/FII14/rtl8188eus
cd rtl8188eus
chmod +x instal.sh
./instal.sh
```
